steps

1. git clone https://github.com/aaribanwar/AI-Sound-Edge-Devices
2. git pull
3. git add .
4. git commit -m 
5. git push

6. cd frontend
7. npm install 
(if node is not present/ npm not recognised)
8. install node from official website
9. restart vs code
10. node -v: verify install
11. npm install : installs everything from package.json automatically
